# uki-uki-diasuki
## 切資料
### 輸入方法
```
python3 cut.py [資料名稱] [切幾份]
e.g. python3 cut.py 2303.csv 12
```
資料會放在data_xxx的資料夾內，如果要重新切資料，記得把舊的資料夾刪掉（我懶得寫刪資料了）
